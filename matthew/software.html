MATTHEW EMMETT

Software projects

I maintain (or co-maintain) the following software projects:

  • PFASST++- a C++ implementation of PFASST.
  • SDCLib- a C implementation of multirate and multilevel SDC scheme (C and Fortran callable).
  • LIBPFASST- a Fortran implementation of PFASST.
  • PyPFASST- a Python implementation of PFASST.
  • PyWENO- a Python implementation of WENO approximations, symbolic WENO functions, and framework for code generation.
  • PyDendro- a set of Python tools for working with and analysing dendrochronological data.
  • PyAsy- a Python module for creating plots with Asymptote.

I have also contributed to a few other open source projects, including:

  • BoxLib - A Fortran and C++ library for block structured codes and finite-volume Adaptive Mesh Refinement (AMR) schemes.
  • PyClaw - A Python based solver for hyperbolic PDEs that includes the algorithms of ClawPack and SharpClaw.
  • FFTW++ - A C++ library (C, Fortran, and Python callable) for computing implicitly dealiased convolutions using FFTW.
